|Python Programming Language| |PyPi| |Supported Python versions| |Development Status| |BSD3 License| |Build Status|

An Enigma machine simulator for Python 2.7.

Overview
========

**Crypto-Enigma** is a Python 2 library for simulating the `Enigma machines`_ most widely used
during the Second World War.

.. _Enigma machines: http://en.wikipedia.org/wiki/Enigma_machine
